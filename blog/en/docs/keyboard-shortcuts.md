# Keyboard Shortcuts

This plugin supports full shortcut customization via calibre.

## How to Customize

1. Open calibre
2. Go to Preferences -> Shortcuts
3. Search for "Ask AI"
4. Edit the shortcuts you want

## Default Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | Ask AI: Ask (global) |
| `F2` | Ask AI: Open Configuration |
| `Ctrl+Shift+L` | Ask AI: AI Search |
| `Ctrl+Enter` | Ask AI: Send (in dialog) |
| `Cmd+Enter` | Ask AI: Send (macOS) |
| `Ctrl+R` | Ask AI: Random Question (in dialog) |
| `Cmd+R` | Ask AI: Random Question (macOS) |
| `Esc` | Close dialog |

> **Note:** If you customized shortcuts in calibre, your custom settings take precedence.
