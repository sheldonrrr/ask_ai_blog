# Installation

## Requirements

- [calibre](https://calibre-ebook.com/) e-book management software (version 5.0 or higher recommended)

## Method 1: Via calibre Plugin Manager (Recommended)

1. Open calibre
2. Go to Preferences -> Plugins -> Get new plugins
3. Search for "Ask AI" (Author: Sheldon)
4. Click Install
5. Restart calibre

## Method 2: Manual Installation

1. Download from https://www.mobileread.com/forums/showthread.php?p=4547077
2. Go to Preferences -> Plugins -> Load Plugin from file
3. Select the .zip file
4. Restart calibre

## Verify Installation

After successful installation, you should see the Ask AI icon in the calibre toolbar.
