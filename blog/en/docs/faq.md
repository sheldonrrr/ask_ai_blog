# FAQ

## Installation Issues

### Q: Plugin not showing in toolbar after installation?

**A:** Try these steps:
1. Confirm you've restarted calibre
2. Check Preferences -> Toolbars & menus to ensure the plugin is enabled
3. Try reinstalling the plugin

### Q: Version incompatibility error?

**A:** Make sure calibre version is 5.0 or higher. Download the latest version from [calibre website](https://calibre-ebook.com/).

## Usage Issues

### Q: AI responses are slow?

**A:** Possible reasons:
- Unstable internet connection
- High server load
- Request content too long

Suggestion: Try simplifying your question or try again later.

### Q: How to switch AI models?

**A:** You can select different AI models in the plugin settings.

### Q: Which e-book formats are supported?

**A:** All formats that calibre can recognize, including:
- EPUB
- MOBI
- PDF
- AZW3
- TXT
- And more
