# 免费 API 密钥选项

## 默认选项: Nvidia AI 免费版 (无需设置)

- **费用:** 完全免费
- **设置:** 无需设置！安装插件后立即可用
- **限制:** 共享免费额度 (高峰期可能不太稳定)
- **说明:** 这是插件开发者维护的免费服务。如需更好的稳定性和更高的限额，请考虑以下选项。

## 选项 1: Nvidia (推荐，更稳定)

- **费用:** 免费
- **限制:** 每分钟 40 次请求
- **设置:** 访问 https://build.nvidia.com/ 并注册
- 获取你的 API 密钥并粘贴到插件的 Nvidia AI 配置中

## 选项 2: Google Gemini

- **免费额度:** 每分钟 15 次请求
- **获取密钥:** https://aistudio.google.com/

## 选项 3: OpenRouter

- **免费模型:** 有一些免费模型可用: https://openrouter.ai/models?q=free
- **获取密钥:** https://openrouter.ai/settings/keys

## 选项 4: Ollama (本地运行，无需网络)

- **费用:** 永久免费
- **隐私:** 在你的电脑上运行，不发送任何数据到网络
- **设置:**
  1. 从 https://ollama.com/ 下载并安装
  2. 在终端运行: `ollama pull llama3.2`
  3. 在插件中选择 Ollama，无需 API 密钥
