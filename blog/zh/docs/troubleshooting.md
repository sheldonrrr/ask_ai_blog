# 故障排除

## 请求失败

- 检查网络连接
- 验证 API 密钥是否正确
- 尝试其他 AI 提供商

## API 密钥无效

- 复制完整密钥，不要有空格
- 检查密钥是否过期
- 对于 Grok: 使用 console.x.ai 的密钥，不是 x.com

## Ollama 不工作

1. 确保 Ollama 已安装并运行
2. 拉取模型: `ollama pull llama3.2`
3. 检查模型: `ollama list`
4. 确认服务运行: `ollama serve`

## 常见问题

### 插件不显示在工具栏

1. 安装后重启 calibre
2. 检查 首选项 -> 工具栏与菜单
3. 确保插件已启用

### 响应很慢

- 尝试其他 AI 提供商
- 检查你的网络连接
- 考虑使用更小/更快的模型

### 模型列表不加载

- 验证你的 API 密钥是否正确
- 检查 Base URL 是否正确
- 尝试再次点击 "加载模型列表"
