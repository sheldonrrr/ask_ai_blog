# 隐私

## 发送给 AI 的数据

- 书籍元数据 (书名、作者、出版商、日期、语言)
- 你的问题
- 你的人设 (如果启用)
- 自定义的提示词

## 不会发送的数据

- 你的 calibre 标签、评论或书评
- 阅读进度
- 个人数据

## API 密钥存储

- 存储在你电脑本地
- 不会发送给第三方
- 位置: `calibre 配置文件夹/plugins/ask_ai_plugin.json`

## 最大隐私

要获得最大隐私，使用 **Ollama**：

- 在你的电脑本地运行
- 不发送任何数据到网络
- 完全控制你的数据

查看 [免费 API 选项](./free-api-options#选项-4-ollama-本地运行无需网络) 了解 Ollama 设置。
