# 基本使用

## 如何使用

1. 在 calibre 中选择一本或多本书
2. 打开 Ask 对话框:
   - 点击工具栏中的 Ask AI Plugin 按钮，或
   - 使用全局快捷键 (默认: `Ctrl+K`)
3. 输入你的问题
4. 点击发送 (或使用 `Ctrl+Enter` / `Cmd+Enter`)

插件会自动包含书籍元数据。你不需要输入书名或作者。

## 主要功能

### 📖 书籍问答

选择书籍后，你可以向 AI 提问关于书籍的问题：

- 书籍摘要
- 内容分析
- 关键概念解释

### 🔍 AI 搜索

使用 AI 搜索书库中的相关内容。

查看 [AI 搜索](./ai-search) 了解详情。

### ✍️ 内容生成

- 生成书评
- 创建阅读笔记
- 提取关键信息
