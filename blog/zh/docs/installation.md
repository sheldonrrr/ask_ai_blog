# 安装指南

## 系统要求

- [calibre](https://calibre-ebook.com/) 电子书管理软件 (建议 5.0 或更高版本)

## 方法 1: 通过 calibre 插件管理器安装 (推荐)

1. 打开 calibre
2. 进入 首选项 -> 插件 -> 获取新插件
3. 搜索 "Ask AI" (作者: Sheldon)
4. 点击安装
5. 重启 calibre

## 方法 2: 手动安装

1. 从 https://www.mobileread.com/forums/showthread.php?p=4547077 下载
2. 进入 首选项 -> 插件 -> 从文件加载插件
3. 选择 .zip 文件
4. 重启 calibre

## 验证安装

安装成功后，你应该能在 calibre 工具栏看到 Ask AI 图标。
