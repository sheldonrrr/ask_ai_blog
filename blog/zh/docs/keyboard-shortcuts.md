# 快捷键

本插件支持通过 calibre 完全自定义快捷键。

## 如何自定义

1. 打开 calibre
2. 进入 首选项 -> 快捷键
3. 搜索 "Ask AI"
4. 编辑你想要的快捷键

## 默认快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` | Ask AI: 提问 (全局) |
| `F2` | Ask AI: 打开配置 |
| `Ctrl+Shift+L` | Ask AI: AI 搜索 |
| `Ctrl+Enter` | Ask AI: 发送 (对话框内) |
| `Cmd+Enter` | Ask AI: 发送 (macOS) |
| `Ctrl+R` | Ask AI: 随机问题 (对话框内) |
| `Cmd+R` | Ask AI: 随机问题 (macOS) |
| `Esc` | 关闭对话框 |

> **注意:** 如果你在 calibre 中自定义了快捷键，你的自定义设置优先。
